Read the instructions carefully
------------------------------------------------

1. Set the path to 1.6JDK(64bit) as Type-1 driver is used.

2. Create a DSN in MS-Access - Follow these steps

Control Panel --> Administrative tools --> Select Odbc Data source (64bit) --> System DSN --> add a new driver --> Select the Database Driver (Microsoft Access Driver (.mdb)) --> Finish --> Create a new database --> Select the drive --> Select the folder --> Give the Database name as Video --> Ok (2) --> Datasource Name --> give a dsn name (video)--> OK(2)

3. Compile and Run Video.java program
